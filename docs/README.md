# BatteryRecorder 使用说明

BatteryRecorder 是一个专注功率记录&续航预测的 App。
致力于以最低的开销实现精准功耗记录，并实现最准确的续航预测！

## 你能在这里看到什么

- 如何开始记录
- 什么时候需要做校准
- 如何排除不想纳入预测的高负载 App
- 为什么续航预测可能暂时没有结果
- 历史记录列表与记录详情页怎么操作

## 你或许想了解

- 先启动服务，App 才会开始记录数据。
- 如果你发现功率显示不准，优先看“校准说明”。
- 如果你发现息屏状态判断异常，去设置页的“服务”分组开启“轮询获取息屏状态”。
- 续航预测至少需要 `3` 条有效记录才可能出结果，样本越多越稳定。
- 首页续航预测卡片可以点击进入详情页。
- 历史记录列表右滑导出、左滑删除。
- 记录详情页底部前三个图例可以点击；功率图例会在 `Raw -> Fitted -> Hidden` 之间切换，电量和温度图例可隐藏。

## 设置页分组

当前设置页主要分成 4 组：

- 常规
- 服务
- 日志
- 预测

和续航预测相关的配置，主要集中在“预测”分组。  
功率显示与校准相关的配置，主要集中在“常规”分组。
